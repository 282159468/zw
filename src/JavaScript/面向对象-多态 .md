不同对象实现某一个功能提供统一接口

## 对象

```js
const username = {
  validator: () => {
    // 验证用户名
  },
};

const password = {
  validator: () => {
    // 验证密码强度
  },
};

const validatorFormItem = formItem => {
  formItem?.validator();
};
```

## 抽象类

```ts
// 抽象类
abstract class Animal {
  eat() {
    console.log('eat');
  }
  abstract sleep(): void;
}

// 无法创建抽象类的实例。
// let animal = new Animal();

// 派生类
class Dog extends Animal {
  constructor(name: string) {
    super();
    this.name = '1111';
  }
  name: string;
  // 子类必须实现抽象类的抽象方法
  sleep() {
    console.log('dog sleep');
  }
}

let dog = new Dog('111');
dog.eat();
```
